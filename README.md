## Steps to run an application:

Download the code from GitHub
<br>
Open the project folder and if possible an editor
<br>
Open the terminal
<br>
Run: 
```
npm install
```
Run:
```
npm run serve
```

Note: Make sure you have installed nodejs and the npm CLI on your machine
